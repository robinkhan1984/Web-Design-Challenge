Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get close to the equator. To accomplish this analysis, we first pulled from the OpenWeatherMap API to assemble a dtaaset of over 500 cities.

After assembling the dataset, we used Matplotlib to plot the various aspects of the weather vs latitude. Factors we looked at uncluded: temperature, cloudiness, wind speed, and humidity. This site provies the source data and visulizations created as posrt of the analysis as well as explanations and descritpions of any trends and correlations witnessed.
